Alice's Adventures Under Ground Project

Tyler Gaydos, Madison Sciarrillo, Charlie Vazquez, Matthew Weitzel

Alice's Adventures Under Ground was the precursor to Alice in Wonderland written by Lewis Carroll. The goal of this project was to preserve and cut down on unnecessary ramblings or deviations from the actual plot of the story. We have marked these sections as "yap" and made a seperate version of the text which has been stripped of these deviations and made the text much shorter and to the point. A color coded view of this text was also made to highlight characters, figurative language, and any markings from the original source.

This website is a continuation of the original project which can be found here. This website reworks the original page using an 11ty framework and integrated javascript to reduce the amount of pages needed, allowing for a smoother and seamless experience.
